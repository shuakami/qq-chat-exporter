# NapCat WebUI

## 功能

- WebUI登录
- QQ登录
- 网络配置
- OneBot/WebUI配置
- 日志查看（实时日志、历史日志）
- HTTP调试
- WS调试
- 在线音乐播放器，支持网易云音乐歌单（大屏在页面右下角，小屏在页面下方）

如果你有更多功能需求，欢迎在 issue 中提出。
