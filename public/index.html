QQ Chat Exporter V4 - 使用文档

欢迎使用全新的 V4 版本！这是一款深度集成到 NapCatQQ 框架的聊天记录导出工具，提供现代化的 Web 界面和强大的导出功能。

V4 版本核心特性

架构升级
深度集成：作为 NapCatQQ 的一部分，共享底层连接，更稳定高效。
现代化 Web 界面：基于 React 和 Next.js 构建，提供直观的图形化操作体验。
API 驱动：提供完整的 RESTful API 和 WebSocket 接口，支持实时进度更新。
持久化存储：采用基于 JSONL 的文件存储系统，支持断点续传和任务恢复。
定时任务系统：支持 cron 表达式，可配置灵活的自动备份计划。
多格式导出：支持 HTML、JSON、TXT 三种格式，满足不同场景的需求。

使用步骤

1. 下载并启动 NapCatQQ

V4 版本已完全集成到 NapCatQQ 中，您只需下载包含 QCE 功能的最新版本即可。

访问 项目 Releases 页面 下载适用于您操作系统的压缩包。
解压后，在 Windows 上运行 launcher-user.bat，在 Linux/macOS 上运行 ./napcat-launcher.sh。
程序启动后，控制台会显示一个二维码，请使用您的手机 QQ 扫码登录。
登录成功后，程序将在后台运行并启动 Web 服务。
2. 访问 Web 管理界面并登录

登录成功后，即可访问 NapCatQQ 的 Web 管理界面：

在浏览器中打开 NapCatQQ 的主管理界面：http://localhost:40653。
如果部署在服务器上，请确保防火墙已放行 40653 端口，然后通过 http://服务器IP:40653 访问。
首次访问时，请根据页面提示，输入控制台中显示的**访问令牌 (Access Token)** 以完成身份验证。
3. 进入 QQ 聊天记录导出模块

登录后，在 NapCat 管理界面的左侧菜单中，找到并点击“QQ 聊天记录导出”即可进入 V4 的功能主界面。

该模块的直接访问地址为：http://localhost:40653/qce-v4-tool。
4. 开始导出任务

在 V4 界面中，您可以通过以下功能模块完成导出：

概览：查看系统状态、任务统计和 NapCatQQ 连接状态。
会话：浏览好友和群组列表，选择要导出的聊天对象。
任务：创建、监控和管理导出任务，支持下载和删除。
定时：设置自动化的定时导出任务，支持每日、每周或自定义周期。
聊天记录：浏览和管理已导出的文件。

注意事项和使用技巧

数据安全

  • 所有聊天记录的获取和导出过程都在您的本地计算机上完成，数据不会上传到任何服务器，确保您的隐私安全。
  • 导出的文件也默认保存在您的本地硬盘上。

使用技巧

  • 首次使用：建议先进入“会话”页面，点击“刷新列表”以确保获取到最新的好友和群组信息。
  • 大批量导出：对于包含大量消息的会话，导出可能需要较长时间。您可以在“任务”页面观察实时进度。任务在后台运行，您可以关闭浏览器标签页，但不能关闭运行中的 NapCatQQ 程序。
  • 定时备份：对于重要的聊天记录，强烈建议使用“定时”功能设置每日或每周的自动备份，以防数据丢失。
  • 格式选择：推荐使用 **HTML** 格式，因为它能最好地还原聊天界面，方便阅读。**JSON** 格式适合需要进行二次开发或数据分析的用户。

🔧 故障排除

  • 无法访问 Web 界面：请检查 NapCatQQ 程序是否已成功启动并登录。同时，请检查 40653 端口是否被其他程序占用。
  • 无法获取好友/群组列表：请确保您的 QQ 账号在线，并在 Web 界面中尝试点击“刷新列表”。
  • 任务失败：任务失败通常会有错误提示。请检查 NapCatQQ 的控制台日志或 Web 界面上的错误信息以定位问题。
